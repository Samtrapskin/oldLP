Project Info

I built an app in React with Redux to help my wife manage her expenses and claims for her massage business. I implemented Material UI tables which helped me create a clean and simple User Interface. Other technologies used: Node.JS, Express, SQL, Bootstrap, Passport, HTML and CSS.

This was also another project I didn't have time to finish. Below are some things I eventually want to add to make this an all-in-one solution:

  • - Make this a mobile frinedly app so that it can be utilized on the go.
  • - Add a scheduling solution so clients can book appointments and the admin can check their schedule.
  • - Incorporate the Square API so the user can transact payments from the app.
  • - Incorporate the Pandora API so that needed music for appointments can be accessed within the app.

Tools

React, Redux, Node.JS, Express, SQL, Bootstrap, Passport, HTML5, CSS and Material UI
